# Planet Facts

A planet facts site.

## Tech Stack

- [SvelteKit](https://kit.svelte.dev)
- [TailwindCSS](https://tailwindcss.com)

## Run Locally

Clone the project

```bash
git clone https://github.com/ashalfarhan/planet-facts.git
```

Go to the project directory

```bash
cd planet-facts
```

Install dependencies

```bash
npm install
```

Start the server

```bash
npm run dev
```
